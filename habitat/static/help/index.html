Map of Life demo release - Help page

This video demonstrates how to use the species list tool works and how to do taxonomic searches, which are the two main methods of getting results.  Map of Life will work on all current versions of most browsers, but we cannot guarantee browser backwards compatibility (e.g. we expect that Internet Explorer 8.0 may not work, but IE9.0 should).  

Searching and selecting species

Figure 1. The Search Widget

  1. You may search for any species by scientific name (e.g. Ochotona princeps) or by vernacular name (e.g. American Pika)
  2. Searching uses auto-complete and you can click on any listed species to return its distribution layers
  3. You can also continue typing and press “Go” get all matching distribution layers (e.g. all with “Pika” or “Ochotona” in name)
  4. After search, a list of distributional product layers appears showing Names, Sources, Types and Results
  5. The two icons next to the species name represent layer source and type and roll-over provides a verbal description
  6. Among product layers returned in “Results”, you can filter by clicking a specific Source or Type
  7. Select any single, set or “all” of product layers in “Results” for display and click “Map Selected Layers”

Working with the Active Layers Interface

Figure 2. The Active Layers Widget

  1. Selected distribution layers are “moved over’’ to your active layers, on the top right of the mapping UI.  
  2. Each layer has an “opacity slider” that you can adjust, along with three other icons:
  1. an eye to make the layer visible or invisible
  2. the letter “z” to zoom to the geographic extent of the layer
  3. the letter “x” that closes the layer.
  1. You may select a layer by clicking it and should see an outline in red around that layer.  Now click any place on the map where that type of distributional product is visible.  
  1. Example: if “regional checklist” is selected, clicking on a region with species presence will retrieve region-specific information.
  1. The active layers can be dragged up or down in the layer stack using your mouse.  Doing so changes the layer ordering.
  2. You can toggle all layers to become invisible or remove all layers at once using the buttons on the “active layers” window.

Species list tool

Figure 3. Species list tool settings

  1. The species list tool will show a list of taxa in an area of interest.  
  2. The tool is toggled on by default. Click the on/off button to toggle, or click the arrow to close.
  3. Select search radius, taxonomic class, and data source (for Reptiles only), and then click  anywhere on the map.  
  4. A list of results is returned in a pop-up window which includes: the detailed coordinates of the search location, the total number of species found, information from wikipedia about each species, a link to the Encyclopedia of Life entry for the species, a gallery of species images, a chart showing species by IUCN status, and a csv download link.
  5. Distribution layers for any species can be mapped right from the list of species or the image gallery.
  6. The results table can be sorted by all columns.
  7. Clicking on Species, Family, and Order names opens the respective Wikipedia pages.

Interacting with the Maps

  1. When a layer is selected in the Layers widget, click on appropriate map elements to retrieve metadata about that layer (for example, if species occurrence points is selected, click on any point to get its description)
  2. You can pan by grabbing and dragging the map and zoom by double-clicking or using the mouse wheel or a trackpad.
  3. You can change the base map using the widget on the lower left of the map application.

Dashboard

  1. The dashboard provides a list about about all of the products in the Map of Life, as well as summary statistics.
  2. Click dashboard entries for complete collections-level information about the products.  
  3. The dashboard list can be filtered by any column, simply enter a search phrase in text box at the top of the column.
  4. All of the dashboard list columns can be sorted  

Legends

  1. Legends are found to the left of layer entries. Click the legend buttons to edit the style of the layer.  
  2. Different parts of the bird range maps represent different portions of migrant birds breeding or non-breeding ranges during summer or winter.
