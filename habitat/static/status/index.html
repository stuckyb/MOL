The Map of Life is provided here in demonstration form. As such, many elements are a work in progress. The following are known issues that you may encounter while using the site. If you find other errors or have suggestions, please provide feedback using the button on the left sidebar.

Known issues in the Map Of Life Demo Release

  1. Browser support for Internet Explorer requires the Chrome Frame Plugin below IE version 9. Chromium / Google Chrome, Safari, and Firefox are the most stable browsers to use with the site.
  2. Map click metadata info windows may respond intermittently. Zoom in or out if this happens.
  3. Google Chrome (all platforms) does not display more than 6 layers at a time.
