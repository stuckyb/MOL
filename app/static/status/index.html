The Map of Life is provided here in demonstration form. As such, many elements are a work in progress. The following are known issues that you may encounter while using the site. If you find other errors or have suggestions, please provide feedback using the button on the left sidebar.

Known issues in the Map Of Life Demo Release

  1. [5/14/2012 11:55 EDT] A large number of species are currently offline. Please stand by.
  2. Browser support for Internet Explorer is limited to version 9. Chromium / Google Chrome, Safari, and Firefox are the most stable browsers to use with the site.
  3. The Species List Tool (right click action) is temporarily exhibiting slow response times due to hardware issues at our database provider, CartoDB. We expect this to be resolved soon.
  4. Hide/show toggle buttons for some range maps are not working in Google Chrome. To turn a layer back on that is not showing, use its opacity slider tool.
  5. Map click metadata info windows may respond intermittently. Zoom in or out if this happens.
